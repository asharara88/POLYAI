# Client-specific playbooks

Use this folder for plays that are tuned to *this client* and override or extend the vertical defaults.

Examples:
- `lead-scoring.md` — this client's specific lead-scoring rules
- `outbound-sequences.md` — sequences tuned to this client's audience
- `objections.md` — the specific objections their buyers actually raise
- `references.md` — case studies / testimonials they're allowed to use

Keep each play short. If a play stops being used, move it to a `archive/` subfolder rather than delete — failed plays are signal.
