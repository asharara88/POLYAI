# Worked examples

These are demonstration clients used to show how the team operates end-to-end. They're not real engagements — treat them as reference, not as live work.

```
_examples/
  meridian-residences/   ← real-estate, off-plan luxury, GCC
```

If you're trying to understand how the system fits together — how onboarding flows into a populated workspace, how `client-profile.md` shapes routing, how a campaign brief gets decomposed into creative briefs — read through one of these end-to-end.

Real client engagements live alongside `_examples/` (without the leading underscore), one folder per client slug. The `_examples/` and `_template/` folders are the only ones in `clients/` that are not real engagements.
